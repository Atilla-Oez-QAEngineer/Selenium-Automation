# Selenium Automation Demo

Ein einfacher automatisierter Test mit Python + Selenium.

## Funktion:
- Öffnet https://demoqa.com/text-box
- Füllt Name, Email, Adresse aus
- Scrollt automatisch zum Submit-Button
- Klickt auf Submit
- Validiert den Output
- Macht einen Screenshot (result.png)

## Dateien:
- **src/test_textbox.py** → Der komplette Testcode
- **screenshots/result.png** → Screenshot des Testergebnisses

## Ausführen:
```bash
python test_textbox.py
